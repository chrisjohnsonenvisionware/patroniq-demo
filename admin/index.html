About this mock

Interactive mock of the new PatronIQ admin console (PRODUCT-SPEC R12–R15), themed for Pioneer Library System per DESIGN-SYSTEM.md. Usage figures are the shared sample dataset (workbench-data/pioneer-data.js) and are marked SAMPLE.

Functional here: role-gated nav, the Events editor (persists to localStorage), module enable stepper, QR generate + signage preview, live AA contrast checks, alert acknowledge, and this notes dock (shared piq_notes key with the backend console).

Sibling: backend/index.html — the API console.
